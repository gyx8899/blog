Steps

  1. Download & Install Git

  2. Download Github Desktop

  3. 使用 SSH 连接到 GitHub

    • 关于 SSH
    • 检查现有的 SSH 密钥
    • 生成新的 SSH 密钥并添加到 ssh-agent
    • 新增 SSH 密钥到 GitHub 账户
    • 测试 SSH 连接
    • 使用 SSH 密钥密码

  4. Clone repo 并测试提交改动，验证提交通过，无需输入账号密码

    • modify file
    • git add .
    • git commit -m ""
    • git push
