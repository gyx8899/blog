# DOS command

## 常见内部命令

```shell script
# change directory 
cd directoryName

# show directory content
dir

# create directory
md directoryName

# remove directory
rd directoryName

# copy dir/file A to dir/file B
copy A B

# delete dir or file
del directoryName
del directoryName/fileName
del fileName

# clear screen
cls

# re-name dir/file
ren 

# copy dir/file
- COPY directoryName
- COPY fileName

# 改变当前盘符
c:
d:
e:

# .bat 执行闪退情况，添加 pause，提示：请按任意键继续...
pause
```
